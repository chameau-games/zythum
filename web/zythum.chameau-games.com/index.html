Comment réussir à s’échapper d’une base secrète en basant toute la réussite de l’opération sur la communication ?


Un bol de CONCENTRATION,

un peu de MÉTHODOLOGIE

et beaucoup de CALME !


C’est ce qu’il vous faudra dans Zythum.

Zythum est un jeu asymétriquement coopératif à deux joueurs contre l’ordinateur.

Les joueurs sont prisonniers d’une base militaire russe. Le but du jeu est de s’échapper de cette base. Pour se faire, un des deux joueurs devra s’infiltrer dans la salle de contrôle afin d’aider l’autre joueur qui, lui, devra se déplacer dans la base pour réaliser différentes missions. Tout cela dans le but final d’ouvrir la porte de sortie sans se faire repérer par les gardes pour enfin gagner la partie !

Zythum est basé sur la communication entre les joueurs, sans celle-ci, ils ne pourront pas gagner.

Zythum est un jeu développé de A à Z par le studio Chameau Games
composé de Léo Arpin, Nathan Goetschy, Adrien Guinard et Rémi Jeulin.

L’envergure du projet était une première pour les quatre membres du groupe. Ce jeu est le résultat d'un projet scolaire réalisé sur une période de 6 mois à l'EPITA en 2021.
Voici le cahier des charges, deux rapport intermédiaire ainsi que le rapport final de projet. Ces rapports racontent l’aventure que l’équipe a vécue tout au long de la création de Zythum et toutes les péripéties rencontrées. Les fonctionnalités du jeu et le processus de réalisation y sont présenté.
